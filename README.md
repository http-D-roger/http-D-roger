## Projects

### Tomato Leaf Disease Classifier — Handheld Edge Diagnostic Device
`YOLOv8s` `MobileNetV3` `ONNX Runtime` `Raspberry Pi 4B` `OpenCV` `TFT Display`

Two-stage computer vision pipeline: YOLOv8s localizes diseased leaf regions, and a MobileNetV3 classifier performs fine-grained diagnosis on cropped patches — all running on a Raspberry Pi 4B with no cloud dependency. Deployed via ONNX Runtime for low-latency on-device inference. Integrated a TFT display and camera input to deliver a fully self-contained, portable field diagnostic tool.

---

### [Smart AI Glasses](https://github.com/http-D-roger/smart-ai-glasses) — OCR-Based Assistive Reading Aid
`Raspberry Pi Zero W` `EasyOCR` `pyttsx3` `OpenCV` `Bluetooth Audio`

Wearable assistive device that captures printed text through a camera and converts it to speech — entirely offline on a Raspberry Pi Zero W with just 512 MB RAM. A capacitive touch sensor triggers capture; audio is routed through Bluetooth earbuds or a wired speaker. Optimized for minimal latency and resource-constrained operation, ensuring full privacy with zero internet dependency.

---

### [Smart Environment Monitoring System](https://github.com/http-D-roger/smart-environment-monitoring) — IoT Sensor Network
`ESP8266` `DHT11` `MQ Sensors` `Blynk` `MQTT`

Multi-sensor IoT system for real-time environmental data acquisition — temperature, humidity, and gas levels — with distributed edge processing for local data handling. Integrated Blynk cloud connectivity for live dashboards, remote alerts, and monitoring. Designed for reliability with optimized sensor interfacing and efficient wireless data transmission.
