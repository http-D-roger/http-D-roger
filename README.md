# Projects


## Tomato Leaf Disease Classifier — Handheld Edge Diagnostic Device

`YOLOv8s` `MobileNetV3` `ONNX Runtime` `Raspberry Pi 4B` `OpenCV` `TFT Display`

Two-stage computer vision pipeline for real-time tomato leaf disease diagnosis on edge hardware. YOLOv8s performs disease region detection while MobileNetV3 provides fine-grained classification on detected regions.

The complete system runs locally on a Raspberry Pi 4B using ONNX Runtime without cloud dependency. Integrated camera input and TFT display provide a portable field diagnostic device capable of performing low-latency disease identification in agricultural environments.

## [Tomato Trait Prediction Using SNP Markers](https://github.com/http-D-roger/tomato-trait-prediction) — AI-Based Genomic Prediction System

`GWAS` `XGBoost` `SHAP` `SNP Analysis` `Bioinformatics` `Streamlit`

AI-based genomic prediction framework that predicts tomato fruit weight using genome-wide SNP marker information.

The project integrates Genome-Wide Association Study (GWAS) for identifying trait-associated genetic markers, XGBoost regression for SNP-based trait prediction, and SHAP explainable AI for understanding the contribution of individual SNP markers.

The pipeline analyzes 7,699 SNP markers from tomato Backcross Inbred Line (BIL) populations, identifies significant genomic regions through Manhattan and QQ plot analysis, and provides biological interpretation by mapping important SNP regions to candidate genes.

Developed an interactive Streamlit dashboard for plant selection, trait prediction, GWAS visualization, and SNP importance analysis, enabling explainable AI-driven genomic selection.

## [Smart AI Glasses](https://github.com/http-D-roger/smart-ai-glasses) — OCR-Based Assistive Reading Aid

`Raspberry Pi Zero W` `EasyOCR` `pyttsx3` `OpenCV` `Bluetooth Audio`

Wearable assistive device that captures printed text through a camera and converts it into speech entirely offline on a Raspberry Pi Zero W with only 512 MB RAM.

A capacitive touch sensor triggers text capture, OCR processing extracts information, and audio output is provided through Bluetooth earbuds or wired speakers. Optimized for low-resource operation with privacy-focused offline processing.

Multi-sensor IoT monitoring system for real-time environmental data acquisition including temperature, humidity, and gas concentration.
